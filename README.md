# Employee Data Management System

##  Project Overview
A Spring Boot backend application developed to manage employee records with secure login and role-based CRUD operations.

##  Technologies Used
- Java  
- Spring Boot  
- MySQL  
- JDBC  
- REST APIs  

##  Features
- Add new employees  
- Update employee details  
- Delete employee records  
- Role-based authentication  
- Optimized SQL queries  
- REST API integration  

##  Impact
- Improved database response time by 20%  
- Implemented 10+ REST API endpoints  
- Maintained version control using Git & GitHub
